Código fonte / Pré-formatação

O comando document.getElementById('Teste') é um comando em Javascript. (usei a tag CODE)

Para criar um cídigo completo com o CODE, basta usar a tag PRE. Segue exemplo:

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print('O número {} é PAR.'.format(num))
    else:
        print('O número {} é ÍMPAR.'.format(num))
